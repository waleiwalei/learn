### 数组扁平化
```js
function getAll(arr){
	let retArr = []
	for(let i = 0; i < arr.length; i++){
		if(Array.isArray(arr[i])) {
			retArr = retArr.concat(getAll(arr[i]));
		} else {
			retArr.push(arr[i]);
		}
	}
	return retArr;
}

```

### 给定数组和条件b 返回符合条件b元素的新数组  要求空间复杂度O(1)
```js
let arr = [1,2,3,46,10,5, 22];
for(let i = 0; i < arr.length;) {
	// if(/*符合条件b*/) {
	if(arr[i] > 5) {
		arr.splice(i, 1);
	} else {
		i++;
	}
}
```
